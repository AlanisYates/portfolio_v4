About Me

Let me tell you about my passion.

Summary

I’m a full-stack developer with a passion for creating clean and efficient websites. This passion has manifested itself in UX/UI design. I’m a firm believer that your website should not only look good it should also run smoothly too. I understand how integral this part of a website can be, and when you combine a great front-end with an equally as great back-end the user experience is complete. To achieve this I use the following technologies.

Front-End

HTML5, CSS3(Sass, CSS Grid, and Flexbox), Bootstrap 4, Javascript(ES6), jQuery, Git, React.js, WordPress, Mocha.js, and Chai.

Back-End

Node.js, PHP, Express.js, Python, MongoDB, SQL, NoSQL, and REST APIs.

Other

Adobe Xd, Adobe InDesign, Photoshop, Lightroom, Figma, Sketch, Matlab, Octave, Microsoft Office, G Suite, Premiere, and Davinci Resolve.
